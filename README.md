# Studio Audio - Sito web

Sito professionale per una società di produzione audio.  
Design system basato su palette nero/oro, tipografia mista (sans-serif per titoli, serif per testi), effetto hover con anteprima audio, player modale e filtri portfolio.

## Struttura
- `index.html` – Home con hero video, highlights e servizi
- `portfolio.html` – Griglia filtrata con hover preview
- `servizi.html` – Quattro pilastri descrittivi
- `team.html` – Tre compositori
- `contatti.html` – Modulo e info

## Tecnologie
HTML5, CSS3 (Flex/Grid), JavaScript vanilla, audio/video HTML5.

## Avvio
Aprire il file `index.html` in un browser moderno.

## Note
Le immagini e i video sono placeholder; sostituire con asset reali.
I file audio di anteprima usano demo SoundHelix (sostituibili con file propri).
